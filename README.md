# Проект: Путешествие по России

### Описание:

Данный проект содержит в себе путеводитель с описанием некоторых регионов для путешествия по необъятной, будоражующей России.

**Инструменты**

* Адаптивная вёрстка;
* Grid Layout;
* Flexbox - вёстка;
* Методология БЭМ;
* Figma;
* HTML5;
* CSS3;
